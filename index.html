Rhetoric English · 상담

오픈채팅은 복사·붙여넣기, 문자는 바로 작성

입력 30초 → 자동 메시지 → 채널별 권장 흐름으로 안내

① 입력폼 작성

학생/학년/목표 등을 입력합니다.

② 자동완성 클릭

입력한 내용으로 상담 메시지가 자동 생성됩니다.

③ 전송 방법 선택

오픈채팅(복사·붙여넣기) 또는 문자(바로 작성)로 보냅니다.

1) 상담 정보 입력

개인정보 저장 없이, 브라우저 내에서만 메시지를 생성합니다.

2) 메시지 자동작성 · 채널 선택

💬 오픈채팅으로 상담 복사·붙여넣기

1) 자동작성 → 2) 복사 → 3) 오픈채팅 열기 → 4) 붙여넣기 → 5) 전송
오픈채팅만 열기
※ 채팅창 자동 입력/전송은 지원되지 않습니다.

✉️ 문자(SMS)로 상담 바로 작성

1) 자동작성 → 2) 문자 앱 열기 → 3) 전송만 탭
전화 걸기
모바일에서 번호/본문이 자동 입력됩니다. (전송 버튼은 직접 탭)

자주 묻는 질문(FAQ)

• 오픈채팅은 보안/스팸 방지로 자동 입력·전송이 제한됩니다. 복사→붙여넣기 흐름을 사용하세요.
• SMS 링크는 모바일 기기에서 동작하며, 일부 기기/브라우저에서는 지원이 다를 수 있습니다.
메시지를 클립보드에 복사했어요.
